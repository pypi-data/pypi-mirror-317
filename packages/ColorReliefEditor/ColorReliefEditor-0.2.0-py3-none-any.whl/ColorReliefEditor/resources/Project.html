This tab creates and opens projects.

Create a New Project

To start a new project, click New. A dialog will appear to create and then open the project folder location. A sample color ramp and settings file will be generated within this folder.

Buttons

Open: Load an existing project previously created by this app.

Open Recent: Access a list of recently opened projects for quick loading.

New: Create a new project, generating all necessary files and folders.

Overview of Tabs

  • Project: Create or open projects. Switch between Basic and Expert mode.
  • Elevation Files: Configure file names and settings for Digital Elevation files.
  • Hillshade: Set hillshade parameters and preview hillshade images.
  • Color: Set color parameters and preview color relief images.
  • Create: Create full-size hillshade and color images and merge them into a relief image.
  • Misc: Rarely changed GDAL settings.(Only in expert/extended mode)
  • Settings: Update general application settings.(Only in expert/extended mode)

Expert Mode

The app features two modes: Basic and Expert.

Basic Mode:
  • Simplified interface with basic settings.
Expert Mode:
  • Offers control over all GDAL settings
  • Supports the use of multiple layers, each with its own elevation files but sharing hillshade and color relief settings.
  • Supports import of GDAL color files using Drag and Drop.
  • Optionally provides two additional tabs for access to every setting by setting "Tabs" to Extended in the Project window.
