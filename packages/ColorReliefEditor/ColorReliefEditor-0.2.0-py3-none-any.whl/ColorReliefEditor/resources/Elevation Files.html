This tab manages settings for Digital Elevation Model (DEM) files and also manages Layers in expert mode.

Instructions

  1. Click on a Download Button to bring you to a website for downloading Digital Elevation files. Follow the instructions under Download Buttons for the specific site.
  2. Once the files are downloaded, drag and drop them into the blue box. This will copy them to the project folder and automatically update the Elevation Files setting.

Download Buttons

Download Elevation (EarthExplorer) - Log in, select Digital Elevation under the Data Sets tab and choose GMTED2010. Use the gmted_mea file.

  • Resolutions
  • 30 arc sec 1000m
  • 15 arc sec 500m
  • 7.5 arc sec 250m

Download US High Res (US National Map) - Log in, select Elevation Products, GeoTIFF format

  • Resolutions
  • 1 arc sec 30m
  • 1/3 arc sec 10m
  • 1 meter 1m

There are numerous other Digital Elevation sources

Settings

  • Elevation Files: This will be set when you drag a file to the blue box. The app will combine multiple elevation files if more than one file is specified.
  • To change the Coordinate Reference System from the default of epsg:3857: switch to expert mode and use the Misc tab.

Expert Settings

In expert mode, you can have multiple layers. A layer has a name and a set of elevation files.

The layers in a project all share the same project settings (color ramp, hillshade settings, etc.)

The elevation files for a layer should be contiguous and have the same resolution.

Running Preview or Create will only build the Active Layer

To create a new layer: just switch Active Layer and type a Layer Name, then add elevation files.

  • Active Layer: Select the layer to use.
  • Layer Name: Specify the name for this layer.
  • Source: (Optional) Provide the source of the elevation files.
  • License: (Optional) Specify licensing information for the elevation files.
